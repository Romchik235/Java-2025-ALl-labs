# Clothing Stream Project

Author: Ð†O-32 Dushko Roman

Description: Generates an infinite stream of clothing domain objects with realistic random values. Gathers 500 items while skipping N occurrences of a chosen city, filters and groups, computes statistics and identifies outliers.

Build and run:
1. Ensure Java 17 and Maven are installed.
2. From project root run: `mvn package`
3. Run the jar: `java -cp target/clothing-stream-project-1.0-SNAPSHOT.jar org.example.clothing.App`
